# [2026-01] 나만의 파이썬 소프트웨어 개발 프로젝트

## 📌 1. 시나리오 제목
👉 대학생 투자 수익 관리 프로그램

---

## 📖 2. 시나리오 내용

최근 주식 투자에 관심이 많아지면서,  
투자 수익을 직접 계산하고 관리할 수 있는 프로그램을 만들고자 하였다.  

이 프로그램은 사용자가 투자금과 수익률을 입력하면 예상 수익을 계산하고,  
여러 투자 데이터를 리스트로 저장하여 관리할 수 있도록 설계하였다.  

또한 단계별로 기능을 확장하여  
단순 계산 프로그램에서 시작해 하나의 투자 관리 프로그램으로 발전시키는 것을 목표로 한다.

---

## 🧩 3. 최종 메뉴 구성 (목표 기능)

1. 투자 추가  
2. 전체 조회  
3. 총 수익 계산  
4. 특정 투자 조회  
5. 투자 삭제  
6. 종료  

---

## ⚙️ 4. V1.0 기능 (2차 과제 목표)

- 사용자로부터 이름, 투자금, 수익률을 입력받는다  
- for문을 사용하여 여러 개의 투자 데이터를 리스트에 저장한다  
- 리스트 데이터를 기반으로 수익을 계산한다  
- 조건문(if~elif~else)을 활용하여 투자 상태를 판단한다  
- 전체 투자 평균 및 총 수익을 계산한다

## 🤖 6. AI 협업 과정

ChatGPT를 활용하여 프로그램 주제를 선정하고,  
투자 수익 관리 프로그램으로 시나리오를 구체화하였다.  

또한 2차 과제의 요구사항에 맞게  
- 리스트 활용 구조 설계  
- 반복문(for)을 이용한 데이터 입력  
- 조건문(if~elif~else)을 통한 투자 상태 판단  

을 중심으로 코드 구조를 설계하였다.  

코드 작성 과정에서는  
입력 데이터 처리, 리스트 저장 방식, 조건문 구성 등에 대해 도움을 받아  
프로그램의 완성도를 높였다.

## 🛠 7. Troubleshooting & 기술 회고

### ❗ 문제 1: 숫자 입력 시 오류 발생
- 문제: 문자열로 입력된 값을 계산할 때 오류 발생  
- 해결: `int()`, `float()`를 사용하여 형변환하여 해결  

---

### ❗ 문제 2: 리스트 데이터 처리 어려움
- 문제: 여러 투자 데이터를 저장하고 계산하는 과정에서 혼동 발생  
- 해결: 리스트에 데이터를 저장하고 `for`문으로 순회하여 해결  

---

### ❗ 문제 3: 조건문 처리 오류
- 문제: 수익/손실 판단이 제대로 되지 않음  
- 해결: `if ~ elif ~ else` 구조를 명확히 정리하여 해결  

---
