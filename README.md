# [2026-01] 나만의 파이썬 소프트웨어 개발 프로젝트

## 📌 1. 시나리오 제목
👉 대학생 투자 수익 관리 프로그램

---

## 📖 2. 시나리오 내용

최근 주식 투자에 관심이 많아지면서,  
투자 수익을 직접 계산하고 관리할 수 있는 프로그램을 만들고자 하였다.  

이 프로그램은 사용자가 투자금과 수익률을 입력하면 예상 수익을 계산해주고,  
여러 투자 정보를 저장 및 관리할 수 있도록 설계하였다.  

또한 단계별로 기능을 확장하여  
단순 계산 프로그램에서 시작해 하나의 완성된 관리 프로그램으로 발전시키는 것을 목표로 한다.

---

## 🧩 3. 최종 메뉴 구성

1. 투자 추가  
2. 전체 조회  
3. 총 수익 계산  
4. 특정 투자 조회  
5. 투자 삭제  
6. 종료  

---

## ⚙️ 4. V1.0 기능 (1차 과제)

- 사용자로부터 이름, 투자금, 수익률을 입력받는다  
- 입력받은 값을 바탕으로 예상 수익을 계산한다  
- 총 금액(투자금 + 수익)을 출력한다  

---

## 🖥️ 5. V1.0 실행 화면 설계


이름을 입력하세요: 예성
투자금을 입력하세요 (원): 1000000
수익률을 입력하세요 (%): 10

===== 투자 결과 =====
이름: 예성
투자금: 1000000원
수익률: 10%
예상 수익: 100000원
총 금액: 1100000원

## 🛠 7. Troubleshooting & 기술 회고

### ❗ 문제 1: 숫자 입력 시 오류 발생
- 문제: 문자열로 입력된 값을 계산할 때 오류 발생  
- 해결: `int()`, `float()`를 사용하여 숫자로 형변환하여 해결  

---

### ❗ 문제 2: 딕셔너리 값 접근 오류
- 문제: 딕셔너리에서 데이터를 가져올 때 오류 발생  
- 해결: `stocks[name]["money"]` 형태로 정확하게 접근하도록 수정  

---

### ❗ 문제 3: 존재하지 않는 데이터 조회 시 오류
- 문제: 없는 이름을 조회하면 프로그램이 정상 작동하지 않음  
- 해결: `if name in stocks:` 조건문을 사용하여 예외 처리
